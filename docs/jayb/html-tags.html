마크업 언어(html) 태그 정리

Jay B 14 Sep 2020 on Devenv

마크업 언어 중 하나인 html의 태그 및 속성을 정리 해본다. html은 웹페이지의 뼈대를 만드는 언어라고 할 수 있다. 프로그래밍 언어라고 분류하기에는 조금 가벼운 감이 있지만 프론트 엔드를 구성함에 있어 아주 중요한 언어이다. 웹페이지는 대부분 html, CSS, Javascript를 사용해 구성되고 있으며 여기서 html과 CSS는 움직임이 없는 화면을 만드는 데 쓰인다.

1. 기본 구성 tags

<!DOCTYPE html>
<html lang=" " dir="ltr">
  <head>
    <meta charset="utf-8">
    <title></title>
  </head>
  <body>

  </body>
</html>
  • !DOCTYPE html : html 문서 유형 (html 5 이후로 거의 현상태로 통일되었다.)
  • html : 웹 페이지 시작과 끝
  • body : head를 제외한 화면에 표현되는 부분
  • head : 상단 영역 혹은 제목 영역으로 화면에 표현되는 부분
  • title : 웹페이지 제목 표시줄에 표현되는 부분

2. 분류 tags

<div>
</div>

<span>
</span>

<pre>
</pre>
  • div : 블록 형식으로 분할 (해당 블록 전체에 동일한 스타일 적용/ 자바 스크립트 블록적용)
  • span : 인라인 형식으로 분할 (텍스트 일부에 스타일 적용/ 자바 스크립트 일부 텍스트 적용)
  • pre : 그대로 출력

  • h1~h6 : 제목 크기별
  • ul : unodered list
  • ol : ordered list
  • li : list
  • br : 줄바꿈
  • hr : 수평줄
  • strong , em : 강조
  • b : 진하게
  • small : 한 단계 작게
  • del : 취소선
  • ins : 밑줄
  • sup : 제곱근 등 표시
  • sub : 밑 등 표시
  • mark : 마크(하이라이트)
  • a : 링크 삽입

4. meta tags

  • meta : 문자인코딩 방식 등
  • base : URL/웹 페이지 출력될 윈도우 설정
  • style : CSS 스타일 시트
  • script : 자바스크립트 코드
  • link : CSS 파일 불러오기 (head에서만 사용)

5. 이미지 삽입 tags

  • img : 이미지
  • src : URL
  • alt : 불러올 이미지가 없을 경우 출력될 문자열
